Running Node Locally for Development
-----------

   * Install [Node.js](http://nodejs.org/)
   * Clone this repo
   * In command line go enter 'cd C:\(your local directorys)\'
   * `npm install sierp`
   * `cd sierp`
   * 'node app.js'
   * Visit `localhost:3000`
